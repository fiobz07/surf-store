# Surf Store

> Ecommerce For Surf store

Products coming from Firebase

## Running Locally

```bash
$ git clone https://github.com/fiobz07/surf-store
$ cd surf-store
$ npm install
$ npm start
```

## Environment Variables

REACT_APP_apiKey=
REACT_APP_authDomain=
REACT_APP_projectId=
REACT_APP_storageBucket=
REACT_APP_messagingSenderId=
REACT_APP_appId=
